Chain Watch Scheduling (In-Dev)
WARNING: This feature is currently in development, therefore certain sub-features may not be fully operational. Contact tiksan for more details.

This feature can generate a possible chain watching schedule based on inputted intervals of availability for faction members. This feature uses a random walk algorithm to generate the chain watching schedule, therefore with each run of the algorithm, different schedules may be outputted. For more information about how this algorithm works, please check this out.
