Tornium Bot

The Tornium bot is a Discord Bot made with Discord.py for Torn City.

For more information, check out the documentation or contact tiksan on Discord. If you want to host this bot on your own server, please follow these the steps listed here.

Features

  • Verification (Requires Implementation) - a verification utility to give user's their Torn name and ID along with faction role and/or position roles
  • Banking - a faction utility for requesting money from the faction vault
  • Stakeouts - a utility to watch factions, users, and companies for changes in specified triggers

Source Code

The Tornium bot is open sourced under the AGPLv3 license. If you decide to fork the bot, you are required to keep the bot under this license. The original source code is developed by tiksan [2383326] and is available on GitHub.
