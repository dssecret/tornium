Bot Documentation

This lists all the possible commands that can be run, but many commands (such as the banking features) are required to be enabled by your server's (herewithin refered to as guild following the Discord naming scheme) administrator(s) or your faction's AA members.

Faction Banking Commands

Faction banking commands include the withdrawal command used by users to request money from the faction bank and the fulfill command used by faction bankers to mark a banking request as fulfilled. NOTE: This set of commands requires the faction AA members to enable faction banking in the faction's bot dashboard.

"?withdraw [amount]"

The faction withdrawal command is used by faction members with cash in the faction vault to send a request to faction bankers to withdraw a certain amount of money. This command checks the faction vault via the Torn API to verify that the requesting user indeed has the requested amount of cash.
